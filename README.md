# Chicagofikr

This is the source code for [chicagofikr.github.io](https://chicagofikr.github.io), a simple website inspired by Dailyfikr.

## How to use

1. Clone or fork this repository to your GitHub account.
2. Edit `index.html` and `style.css` to customize your site's content and appearance.
3. Push your changes to GitHub.  
4. Enable GitHub Pages from your repository's settings.  
5. Visit [https://chicagofikr.github.io](https://chicagofikr.github.io) to see your site live!

## File Structure

```
index.html      # Homepage
style.css       # Styles
README.md       # Project info
```
